# BoxSounds

An e-commerce platform for selling custom vinyls/CDs online.

## Features

* Product catalog.
* Shopping cart.
* Payment system.
* Order management.
* Admin panel.

## Technologies

* Next.js,
* Tailwind,
* React,
* TypeScript.
