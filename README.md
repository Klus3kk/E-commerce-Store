# BoxSounds

An e-commerce platform for selling custom vinyls/CDs online.

## Features

* Product catalog.
* Shopping cart.
* Payment system.
* Order management.
* Admin panel.

## Technologies

* Next.js,
* Tailwind,
* React,
* TypeScript,
* Lucide (for icons).

## Important commands (ignore this)

```bash
pnpm dlx create-next-app@latest

npx shadcn@latest init

npx shadcn@latest add (button for example)
```
